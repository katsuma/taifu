# taifu
taifu will bring YouTube sound to your iTunes library silently.

## Usage

Try

```
taifu http://www.youtube.com/watch?v=I1X6MrBfjrk
```

And check type 'taifu' at search box on your iTunes.


## Install

```
gem install taifu
```

# Supported OS
- OSX

## Dependencies
- ffmpeg
- youtube-dl

You can install these tools by homebrew.
